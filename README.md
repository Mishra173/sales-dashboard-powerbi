Sales Dashboard (Power BI)

 Overview

This project uses Power BI to analyze sales performance and order trends from a sales dataset.
The dashboard helps understand revenue, profit, products, channels, and order processing behavior.

Main columns include:
OrderDate, ShipDate, Country, Region, SalesChannel, ItemType, UnitsSold, TotalRevenue, TotalCost, TotalProfit.

 Dashboard Features
- Cleaned and transformed raw data using power query
- KPIs: Total Revenue, Total Profit, Units Sold, Avg Processing Days
- Sales & Profit by Channel
- Sales & Profit by Country
- Top 5 Products
- Monthly Sales & Profit Trend
- Order Processing Trend (Yearâ€“Month)

Key Insights

* Sales and profit vary across channels and regions.
* Some product categories contribute more to revenue and profit.
* Monthly sales and profit show changing trends over time.
* Order processing times fluctuate and may impact performance.
